# 加载项目上下文

快速了解当前项目的全貌，建立完整的上下文。

## 执行步骤

1. **项目配置**
   - 读取 CLAUDE.md、README.md、package.json、build.gradle 等项目配置
   - 识别技术栈、框架版本、构建工具

2. **目录结构**
   - 列出项目的顶层目录结构
   - 识别关键目录（src、test、config 等）

3. **架构概览**
   - 识别分层结构（controller/service/mapper 等）
   - 识别核心模块和业务领域

4. **关键文件**
   - 入口文件（main、index、App）
   - 路由/API 定义
   - 配置文件
   - 数据库 schema / migration

5. **开发环境**
   - Git 分支状态
   - 最近的提交记录（10 条）
   - 是否有未提交的改动

## 输出

以简洁的结构化格式输出项目概览，帮助快速理解：
- 这是什么项目
- 用了什么技术
- 代码怎么组织的
- 当前开发状态
